# Codal Hub

Scalable software & web applications — built with React + Vite.

## Tech Stack

- **React 18**
- **Vite 5**
- **Google Fonts** (Instrument Serif, Inter)

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Output is generated in the `dist/` folder.

## Preview Production Build

```bash
npm run preview
```

## Deployment

This project is deployed on [Vercel](https://vercel.com).

No additional configuration is required — Vercel auto-detects Vite and uses:
- **Build command:** `npm run build`
- **Output directory:** `dist`
